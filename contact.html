📧 Proton Mail

The easiest way to reach me securely and anonymously, is using Proton Mail, an e-mail service based in Switzerland. In most cases, Protonmail is sufficiently secure. Proton has written a blogpost with use-cases for their service.

If possible, please use the Tor-browser when creating an anonymous Protonmail-account.
After you've done that, send your message to: daanvanmonsjou@protonmail.com

Tip: Protonmail has a separate, optional website specifically for the Tor-browser: https://protonirockerxow.onion/

Do note that you cannot open this link using a 'regular' webbrowser like Chrome or Firefox.
